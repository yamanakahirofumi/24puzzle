# 24-Puzzle ルール

24-Puzzle（24パズル）は、5×5の格子（計25マス）の中に1から24までの数字が書かれたパネルが配置されており、1マスの空きを利用してパネルをスライドさせ、目的の並び順（通常は左上から順に1, 2, 3...）に並べ替えるパズルゲームです。

## 基本ルール
- **盤面**: 5×5の計25マス。
- **パネル**: 1から24までの数字が書かれた24枚のパネル。
- **空きマス**: 盤面には常に1つの空きマスが存在します。

## 操作方法
- 空きマスに隣接するパネル（上下左右）を選択し、空きマスの位置へスライドさせることができます。
- パネルを一度に複数枚動かすことはできず、1手につき1枚のパネルを動かします。
- **解の保証**: シャッフル時には、必ずゴール（勝利条件）に到達可能な配置が生成されます。

## ゲームの開始とシャッフル
- **初期状態**: アプリケーションの起動時やリセット時には、すべてのパネルが正しい位置に並んだ「完成状態」が表示されます。
- **ゲーム開始時の動作**: 「開始」ボタン（またはシャッフルボタン）を押すと、完成状態からランダムにパネルをスライドさせることで盤面がシャッフルされ、ゲームがスタートします。
- **解の保証**: 完成状態から有効な移動を繰り返してシャッフルを行うため、提示されるパズルは常に解くことが可能です。

## 勝利条件
- すべてのパネルを昇順（1, 2, 3, ..., 24）に並べ、右下のマスを空きマスにすることで完成となります。

## 特徴
- 15パズル（4×4）の拡張版であり、より複雑な戦略が必要となります。
- **画像パズル機能**: ユーザーが任意の画像ファイルを読み込み、その画像を分割してパズルとして遊ぶことができます。
    - **補助機能**: 画像パズルでは、各タイルの正解位置を示す数字を画像の上に重ねて表示することができます（設定で切り替え可能）。
