# 数字の仕様

本ドキュメントでは、24-Puzzle における数字の定義、配置、および画像パズル時の表示仕様について定義します。

## 1. 数字の定義
- パズルで使用される数字は **1 から 24** までの整数です。
- 各数字は、5×5 グリッド上の特定の「正解位置」に対応しています。

## 2. 配置とゴール状態
- **ゴール状態（完成図）**:
    - 1行目: 1, 2, 3, 4, 5
    - 2行目: 6, 7, 8, 9, 10
    - 3行目: 11, 12, 13, 14, 15
    - 4行目: 16, 17, 18, 19, 20
    - 5行目: 21, 22, 23, 24, (空きマス)
- 右下のマス（5行5列目）は常に空きマスとなり、数字パネルは配置されません。

## 3. 表示モード別の仕様

### 3.1 通常モード（数字パズル）
- 各パネルの中央に、数字を大きく表示します。
- 背景色と数字の色のコントラストを十分に確保し、視認性を高めます。

### 3.2 画像モード（画像パズル）
- ユーザーが選択した画像の上に、補助情報として数字を重ねて表示（オーバーレイ）します。
- **配置**: 各タイルの**右下**に配置します。
- **視認性確保**:
    - テキストカラー: 白
    - 装飾: 黒のドロップシャドウまたは縁取りを付与し、画像の色に左右されず読み取れるようにします。
    - 不透明度: 画像を過度に遮らないよう、適度な半透明（例: 70%）に設定します。
- **表示制御**:
    - コントロールパネルの「数字表示の切り替え」設定により、動的に表示/非表示を切り替えることができます。
