# 技術スタック

本プロジェクトの開発に使用される技術、ライブラリ、およびツールの一覧です。

## 言語・フレームワーク
- **言語**: Java
- **GUIライブラリ**: JavaFX
  - バージョン: 21

## ライブラリ

- **カバレッジ測定**: JaCoCo (Java Code Coverage Library)

- **ビルドツール**: Maven
  - バージョン: 3.9
- **配布ツール**: jlink, jpackage (JDK 標準)
